# Student Burnout Prediction System

A machine learning-based web application to predict student burnout using study habits and lifestyle factors.

## Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas

## Features
- Predict burnout level
- REST API using Flask
- ML model integration

## How to Run
1. Install dependencies
2. Run `app.py`
3. Use POST request on `/predict`
